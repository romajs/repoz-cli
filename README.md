# repoz-cli

Repoz: [http://repoz.dextra.com.br]()

## Install

`npm install -g repoz`

## Usage

`repoz -p <project> <command> [args]`

## Commands

Each command will prompt user credentials (username & password)

### GET

`get <path> [file]`

### POST

`post <path> <file>`

### PUT

`put <path> <file>`

### DELETE

`delete <path>`

### LIST

`list`
